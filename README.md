# Sveltekit-Electron

1. 完美的开发和发布体验
2. 双 package.json 不引入开发依赖
3. eslint 代码自动检查
4. push 前代码自动格式化
5. 自动化的开发日志
6. 自动同步 package.json 的版本
7. 自动 git 管理

## Quick Start

#### prepare

`pnpm i`

#### dev

`pnpm dev`

#### release

**prerelease**

`pnpm prerelease`

**release**

`pnpm release`
